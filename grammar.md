# 语法总结

# 名词-Nouns

`名词`就是能指代`人`，`事`，`物`，`地点`或`概念`的词语。

名词在句子中可以做`主语`，`表语`，`同位语`，`宾语`，`宾补`。

`专有名词`指特定名称，通常是`人，事，物，地点或概念`的名字，因此首字母大写，如John是一个人的名字，Monday指一星期中特定的一天。

`普通名词`指一般名称，首字母不需要大写，如apple（但是如果是指定苹果手机，应该要大写），school。

`具体名词`指五官能感受到的物质，如table，juice，air等。

`抽象名词`指无法用五官感受的概念，如love，time，childhood等。

`可数名词`指有`单数与复数区别`的名词，如book，flower等。可数名词又可以分为`普通名词`和`集合名词`两种。

`单数普通名词`之前，通常会加`a/an`或`the`；`复数普通名词`之前不加冠词，而是在词尾加上`s`或`es`表示复数。

`集合名词`指同种类的人或物之集合体名称，形式是单数，但代表的却是复数，如family。

`不可数名词`则是无法计数，无单复数区别的名词，如water，light等。不可数名词前一般不加冠词a/an，也没有复数形式。

**单数名词变复数名词的的方法**

（规则变化

* 一般名词词尾加s，如girl-girls
* 词尾为o，加es，如hero-heroes
* 词尾为s，x，sh，ch，加es，如fish-fishes
* 词尾为f或fe，变f或fe为v，再加es，如knife-knives
* 词尾为辅音+y，先变y为i，再加es，如baby-babies
* 词尾为元音+y，直接加s，如boy-boys

不规则变化

* 名词中的元音变化，如man-men
* 名词词尾+en/ren，如child-children
* 单复数形式相同，如sheep-sheep

# 代词-Pronouns

代词，就是`代替名词`的词。

* 人称代词：代替人名或称谓。如we或you
* 指示代词：代替特定人或物。如this或these
* 不定代词：代替不特定人或物。如some或many
* 疑问代词：代替不确定人或物。如who或which
* 关系代词：同时具有代词及连词功能。如which或whose

## 人称代词

代替人名或称谓的人称代词，有单复数的区别，并有主格，形容词性物主代词，宾格三种形式。

|          |      | 主格                | 形容词性物主代词 | 宾格       |
| -------- | ---- | ------------------- | ---------------- | ---------- |
| 第一人称 | 单数 | I                   | my 我的          | me         |
|          | 复数 | we                  | our 我们的       | us 我们    |
| 第二人称 | 单数 | you                 | your 你的        | you        |
|          | 复数 | you 你们            | your             | you        |
| 第三人称 | 单数 | he/she/it           | his/her/its      | Him/her/it |
|          | 复数 | they 他们/她们/它们 | their ta们的     | them ta们  |

主格：I am a student

宾格：She loves me 注：第三人称，中间动词记得变成单三形式。

## 名词性物主代词

名词性物主代词是为了代替`形容词性物主代词+名词`的代词，当前面出现过某个名词时，为了避免重复，后面就会用名词性物主代词代替前面出现的名词。

| my+名词      | mine   |
| ------------ | ------ |
| our + 名词   | ours   |
| your + 名词  | yours  |
| his + 名词   | his    |
| her + 名词   | hers   |
| its + 名词   | its    |
| their + 名词 | theirs |

例：

> Is this you umbrella? 这是你的雨伞吗？
>
> Yes，it is mine

前面的问句已经提到了`umbrella`雨伞这个名词，后面的答句就不需要重复umbrella，改用`mine`代替`my umbrella`。

## 反身代词

当`主语`与`宾语`指的是`同一人或同一件事`时，`宾语`要使用`反身代词`。

|      | 主语 | 宾语 | 反身代词            |
| ---- | ---- | ---- | ------------------- |
| 单数 | I    | me   | myself              |
|      | you  | you  | youself             |
|      | he   | him  | himself             |
|      | she  | her  | hersef              |
|      | it   | it   | itself              |
| 复数 | we   | us   | ourselves           |
|      | you  | you  | youselves           |
|      | they | them | themselves 他们自己 |

例：

> I know me ×
>
> I know myself √

## 指示代词

用来代替`特定的人或事物`的代词。指示代词也可以做指示形容词，后面接名词。

| 单数 | this  | 这个（人，事，物） |
| ---- | ----- | ------------------ |
|      | that  | 那个               |
| 复数 | these | 这些               |
|      | those | 那些               |

指示代词在句中可以做主语，也可以做宾语。

1. 做主语

> This is my seat = 这是我的座位。

2. 做宾语

> 	I don't want to eat that = 我不想吃那个东西。

3. 做指示形容词，后面接名词

> You can't borrow this book = 你不能借这本书。

## 不定代词

用来代替`不特定的人或事物`的代词。不定代词也可以做不定形容词，后面接名词。

| one    | 一个（人、事、物） |
| ------ | ------------------ |
| each   | 每一个             |
| both   | 两个               |
| either | 两者其中之一       |
| any    | 任何、若干         |
| some   | 一些               |
| many   | 很多               |
| more   | 更多               |
| most   | 最多               |
| all    | 所有               |
| none   | 没有               |
| a few  | 几个               |
| a lot  | 很多               |

不定代词在句子中可以做主语，也可以做宾语。

1. 做主语

> I have many bags. `Some` are expensive, but `most` of them are cheap
>
> some指“有些包”，但没有特别指出是哪些包。most指“大部分包”，但没有特别指出是哪些包。
>
> 我有很多包。有些很贵，但大部分都很便宜。

2. 做宾语

> The cookies look yummy. Can I have `one`?
>
> one指“一块饼干”，但没有指定是哪一个。
>
> 这些饼干看起来很美味。我可以吃一块吗。

3. 做不定形容词，后面接名词

> She only has `a few` friends.
>
> 她只有几个朋友

## 疑问代词

疑问代词是用来询问`人、事物、时间、地点、方法、原因`的代词，单复数形式皆相同，常放在句首作为疑问句。

| who 谁       | 问”人“                   |
| ------------ | ------------------------ |
| whom 谁      | 问”人“                   |
| whose 谁的   | 问”持有人“               |
| what 什么    | 问”事、物“或”职业、身份“ |
| which 哪一个 | 问”人、事、物“           |

疑问代词在句子中可以做主语，主语补语，宾语，介词的宾语。

1. 作主语

> Who is in the bathroom？谁在厕所里？

2. 作主语补语

> Whose is that? 那个是谁的?
>
> whose为主语that的补语

3. 作动词的宾语

> Whom do you love ？你爱谁?
>
> whom为动词love的宾语

4. 作介词的宾语

> What are you talking about? 你们在讲什么？
>
> what为介词about的宾语

# 所有格

所有格可放在名词前面，表示该名词是属于`谁的`。正因为放在名词前面，所以也可以视为形容词，或称为`所有形容词`，如his book（他的书）、Amy's boyfrind（艾米的男朋友）。所有格也可以用`介词of`来表示`属于...的`，如the cover of the book（书的封面）。

**所有格的形成方式：**

* 形容词性物主代词 -- my/our/his/its/her/your/their
* 名词+`‘s` -- Peter's/my monther
* of+名词 -- of the table/of London
* 集合名词，后面直接加`’s` -- children's/people
* 词尾为s的复数名词，加`‘`，不加s -- the boys'/the students'
* 两人以上共同拥有，最后一个名词后面加`’s` -- Steven and Jack's
* 两人以上各自拥有，每个名词后面各自加`‘s` -- Steven's and Jack's

**所有格的小陷阱：**

> Peter and Jerry's mon VS Peter's and Jerry's moms
>
> Peter and Jerry's mon 指 彼得和杰瑞两个人是兄弟，两人拥有一个共同的妈妈，所以mon是单数
>
> Peter's and Jerry's moms 指 彼得和杰瑞各自的妈妈，所以mon为复数

**所有格的用法：**

所有格无法独立在句子中使用，后面一定要接名词形成”名词短语“，并作为主语、主语补语（主语补语也就是`表语`）或宾语。（若是of + 名词，则名词放前面）

1. 作主语

> `My father` is a police officer = 我的父亲是一名警官

2. 作主语补语（表语）

> His wife is the `boy's teacher ` = 他太太是男孩们的老师

3. 作动词的宾语

> I can't stand `the weather of London` = 我无法忍受伦敦的天气

4. 作介词的宾语

> I am in love with Amy and Emily's brother = 我爱上了艾米和艾米丽的哥哥

# 冠词-Articles

冠词是一种限定词，`不定冠词`的`a/an`通常放在`未指定`的单数名词前，如a boy；`定冠词`的`the`则放在`有指定`的单复数名词或不可数名词前，如the boy。冠词虽然分”定冠词和不定冠词“两种，但总共只有三个，即”a、an、the“。

**不定冠词：a/an**

a和an都是指一个，因此后面只能接单数的可数名词。

| a    | 用在首字母为`辅音`的单数名词前 | a book/a window/a year   |
| ---- | ------------------------------ | ------------------------ |
| an   | 用在首字母为`元音`的单数名词前 | an egg/an hour/an island |

注意：



**定冠词：the**

the没有单复数之分，后面可以接单数可数名词、复数可数名词，也可以接不可数名词。

**什么时候用定冠词：**

1. 表示`特定的`的人、事、物

有特定指称的对象时，前面要加定冠词the，以表示”指定“。

> Mr.Smith is the teacher of the class = 史密斯先生是这个班级的老师
>
> Did you enjoy the movie? = 你喜欢那部电影吗？

2. 表示独一无二的自然事物

自然界独一无二的事物前面，要加冠词，表示”唯一“

> the moon、the sun、the sky
>
> The earth moves around the sun = 地球绕着太阳转

3. 表示方向或方位时

> the right/the left/the north/the east
>
> The forest is on the right of the river = 森林在河的右侧

4. 专有名词前

在”专属名词“的自然景观，如河流、山脉、群岛、海洋或建筑物等，前面也会加上定冠词the。

> the Pacific Ocean 太平洋、the Nile 尼罗河、the Alps 阿尔卑斯山
>
> She swam across the Yellow River = 她游泳横渡了黄河

5. 序数词前

表示”第几......“的序数词之前，必须加定冠词the。

> the first 第一、the second 第二、the tenth 第十
>
> She was the first one to get up = 她是第一个起床的人

6. 形容词最高级之前

表示”最......“的形容词最高级之前，必须要加定冠词。

> the best 最棒的、the worst 最糟的、the most 最多的、the least 最少的
>
> I have the best mon in the world = 我拥有世界上最棒的妈妈

# 形容词-Adjectives

形容词是用来`修饰”名词“和”代词“`的一类词，其存在与否并不影响句子的结构，但是使用形容词能增加句子的丰富性。

如”I have an idea“和”I have an gread idea“，这两个句子比较起来，后者多了形容词great来修饰名词idea，比前句更具体，更能吸引大家对idea的注意。

**形容词的语法：**

形容词有两种用法，一种是用来放在`名词之前`修饰名词，一种是放在`系动词`之后作宾语。

1. 放在名词之前修饰名词

形容词+名词=名词短语

> smart boy、crazy idea、pretty girl

名词短语为名词的相等语，在句子中可视为”一个名词”来使用，可以作为句子的主语，亦可以作为补语或宾语。

> The handsome man is my husband = 哪个英俊的男子是我的丈夫
>
> Mike is a smart boy = 迈克是个聪明的男孩
>
> She wore a beautiful dress = 她穿了一条美丽的连衣裙

2. 放在系动词之后，作为表语

形容词可以放在系动词如is、are或look、smell等后面，作为修饰前面主语的补语（表语）

> My parents are happy
>
> The cake looks deliciour = 蛋糕看起来很美味

**形容词的种类：**

1. 代名形容词：可以当作形容词用的代词，即代名形容词
2. 数量形容词：用来表示名词的数或量有多少的形容词
3. 性状形容词：用来修饰名词的性质，外观，状态，特性等的形容词

**形容词的顺序：**

> 代名形容词>数量形容词>性状形容词 + 名词

1. 包含“冠词、指示形容词、不定形容词及所有形容词”的代名形容词要放在名词短语的最前面。
2. 包含“序数词、数词”的数量形容词紧接在后。
3. 最后才是放描述“性质、外观、专有形容词、材质”等的形状形容词。

> The first two days = 前两天 -- 冠词->序数词->数词->名词
>
> These two old ladies = 这两个老太太 -- 指示形容词->数词->性状形容词->名词

**性状形容词的顺序：**

性状形容词是离名词最接近的形容词，而用来修饰同一个名词的性状形容词可能不止一个，即使同样都属于性状形容词，也有一定的顺序。

基本上，属于对名词的主观判断，或是描述外观形状的形容词离名词最远，而越能描述名词本质的形容词，则离名词最近。

| 主观                    |                    |           ->            |                |                  | 客观              |
| ----------------------- | ------------------ | :---------------------: | -------------- | ---------------- | ----------------- |
| 特质                    | 外观（大小，形状） |   状态（温度，新旧）    | 颜色           | 国籍             | 材质、材料        |
| friendlly/kind/generous | big/samll/tiny     | clean/dirty/new/old/hot | red/blue/whtie | Chinese/Japanese | stone/silver/wood |

> friendly young American woman = 一位亲切的美国年轻女性
>
> 主观特质->主观状态->客观国籍->名词

## 比较级和最高级

用来修饰名词的形容词，会因为修饰程序不同而有所`比较`。比较可以依程度分为`原级`、`比较级`、`最高级`三种。

形容词的变化方式，可分为`规则变化`和`不规则变化`两种：

1. 规则变化

|                             | 原级      | 比较级         | 最高级          |
| --------------------------- | --------- | -------------- | --------------- |
| 多数单音节的形容词          |           | 加er           | 加est           |
|                             | short     | shorter        | shortest        |
|                             | strong    | stronger       | strogest        |
| 词尾为e的形容词             |           | 加r            | 加st            |
|                             | nice      | nicer          | nicest          |
|                             | wise      | wiser          | wisest          |
| 词尾为辅音+y的形容词        |           | 将y改i，再加er | 将y改i，再加est |
|                             | lazy      | lazier         | laziest         |
|                             | happy     | happier        | happiest        |
| 短元音+单辅音的单音节形容词 |           | 重复词尾加er   | 重复词尾加est   |
|                             | big       | bigger         | biggest         |
|                             | fat       | fatter         | fattest         |
| 两个音节以上的形容词        |           | 前面加more     | 前面加most      |
|                             | helpful   | more helpful   | most helpful    |
|                             | important | more important | most important  |

2. 不规则变化

| 原级      | 比较级 | 最高级 |
| --------- | ------ | ------ |
| good/well | better | best   |
| bad/ill   | worse  | worst  |
| many/much | more   | most   |
| little    | less   | least  |

3. 特殊的形容词比较级

有些形容词有两种不同的比较级和最高级，且代表的含义稍有不同。

| 原级       | 比较级                   | 最高级   |
| ---------- | ------------------------ | -------- |
| old 老；旧 | older 更老的；更旧的     | oldest   |
|            | elder 较年长的；较资深的 | eldest   |
| far 远     | farther 更远的           | farthest |
|            | further 更进一步的       | furthest |
| late       | later 较晚的             | latest   |
|            | latter 后者的            | last     |

**形容词比较级的基本句型**

两人或两物作比较时，两者皆用“主格”。

1. 原级比较

> 主格A+be动词+`as+形容词原级+as`+主格B

这个句型表示“A与B一样....”，在be动词后加not即形成否定句，表示“A不如B那么......”。

> Peter is <u>as tall as</u> his father = 皮特跟他爸爸一样高
>
> This book is not <u>as interesting as</u> that one = 这本书没有那本有趣

2. 比较级比较

> 主格A+be动词+`形容词比较次+than`+主格B

这个句型表示“A比B更......”，在be动词后面加not即形成否定句，表示“A没有B更......”。

> Rock music is <u>nosier than</u> light music = 摇滚乐比轻音乐更吵
>
> Jennifer is <u>not younger than</u> Lisa = 珍妮弗并不比丽莎年轻

## 形容词最高级的基本句型

最高级用在`三者或三者以上`作比较时，最高级形容词前面必须加`定冠词the`或`形容词性物主代词`。

> 主语+be动词+`the/所有格`+`(most)形容词最高级`(+名词)+范围

这个句型用来表示“主语是（某范围）之中最....的”，be动词后面加not即形成否定句。

> Kevin is <u>the smartest</u> student in this class = 凯文是这个班上最聪明的学生
>
> Susan is my dearest little sister  = 苏珊是我最亲爱的妹妹

## 用比较级表示最高级

> 主语+be动词+形容词比较级+than+`any other 单数名词/all the other 复数名词`

* any other 指“任何其他的”，后面接单数名词，表示主语与某范围内的任何一个其他人作比较。
* all the other 指“所有其他的”，后面接复数名词 ，表示主语与某范围内的所有其他人比较。

以下三个句子表达同样的意思：

> Mr Simpson is <u>richest businessman in our town</u> = 辛普森先生是我们镇上最富有的生意人
>
> Mr Simpson is <u>richer than any other businessman</u> in our town = 辛普森先生比我们镇上任何其他生意人都富有
>
> Mr Simpson is <u>richer than all the other businessmen</u> in our town = 辛普森先生比我们镇上所有其他的生意人都富有

# 动词-Verbs

动词就是用来`形容或表示各类动作及状态`的词。

基本上每个完整的句子都有一个动词，要表示`第二个动作`可使用`不定式`、`动名词`、`并列连词`、`从属连词`或`增加子句`等方法连接。

英语动词是句子的核心，它即决定着句子意思的表达，同事又决定了句子的语法结构。

## 实义动词

实义动词意思完整，能独立用作谓语。实义动词又分为`及物动词`和`不及物动词`。

### 及物动词

所谓“及物”，就是主体把动作专递到客体上。换句话说，就是指动词后面必须接一个宾语才能使句子表达意思完整。

及物动词的特征就是本身意义不完整，其后要加上宾语才能表达完整的意思。及物动词的宾语可以是人，也可以是物。

> That dog makes the boy happy = 那条狗使这个孩子高兴起来（宾语是人）

**及物动词句型分类：**

* 主+谓+宾
* 主+谓+双宾
* 主+谓+宾+宾补

### 不及物动词

不及物动词是指后面`不需要宾语而意义完整`的动词。

不及物动词的特征就是本身词义完整，后面不需要加宾语。只能用于：`主+谓`结构。

## 系动词

系动词也叫`连系动词`，有些不具词义，有些具有词义，但不能单独用作谓语，后面必须跟`表语(主语补语)`，构成`系表结构`，来说明主语的`状况、性质快、特征`等情况。

连系动词有三种：

1. be动词

> be=是 动词原型
>
> am(was)=是 （使用与第一人称单数）

2. 描述当下感觉的感官动词

> sound 听起来、look 看起来、smell 闻起来、feel 摸起来；感觉起来、taste 尝起来

3. 表示”转变、转换，变成“的动词

> get 变得、become 成为、turn 变成

## 助动词

组动词顾名思义就是用来表示`帮助`动词形成句子的词。

实义动词想要表示`否定`、`疑问`、或是表示`时态`、`语态`以及增加`语气`，都得靠组动词的帮助，才能达到目的。

1. 帮动词表示否定

> 助动词+not+动词原型

一般肯定句中的动词，不需要组动词的帮助，但是当句子需要表示`负面`、`否定`的含义时，组动词就会带着`not`来帮助动词。

> We `do not` speak English = 我们不说英语

2. 帮动词表示疑问

> 助动词+主语+动词原型

当句子要表示疑问是，需要助动词来帮忙，助动词会放到句首，将整个句子变成疑问句。

> 陈诉句 - They play basketball = 他们打篮球？
>
> 疑问句 - Do they play basketball？ = 他们打篮球吗？



## 情态动词

情态动词是指表示说话人的`语气或情态`的动词。情态动词没有人称或数的变化，在句中不能单独作谓语，必须和后面的`动词原形`合用。

情态动词的否定形式，可以直接在后面加`not`，也可以简写`情态动词't`。

情态动词的一般疑问句形式，可以直接把情态动词提到句首，句中的其他成分不变，并且在句末加问号。

## 动词的时态-Tenses

* 一般现在时

* 一般过去时

* 一般将来时

  

## 动词的时态变化

时态指的就是`时间`和`状态`。

英文与中文最大的差异就在于：中文没有时态，而英文有时态区分。用来区分各种不同时态的，就是句子中的”动词“。

动词的时态变化分为`动词原形`、`动词过去式`，`过去分词`、`现在分词`。

| 状态\时间 | 过去                          | 过去的将来                               | 现在                               | 将来                                |
| --------- | ----------------------------- | ---------------------------------------- | ---------------------------------- | ----------------------------------- |
| 一般      | 一般过去时-did                | 过去将来时-would do                      | 一般现在时-do/does                 | 一般将来时-will do                  |
| 完成      | 过去完成时-had done           | 过去将来完成时-would have done           | 现在完成时-have/has done           | 将来完成时-will have done           |
| 进行      | 过去进行时-was/were doing     | 过去将来进行时-would be doing            | 现在进行时-am/is/are doing         | 将来进行时-will be doing            |
| 完成进行  | 过去完成进行时-had been doing | 过去将来完成进行时-would have been doing | 现在完成进行时-have/has been doing | 将来完成进行时-will have been doing |

1. 动词原型

句子时态为`现在时`，动词要用`动词原型`，如play。

但是如果主语是`第三人称单数`时，动词要变成`单数动词`（动词的单三形式），如plays。

> I <u>play</u> football after school every day = 我每天放学后踢球
>
> He <u>plays</u> football after school every day = 他每天放学后踢足球

2. 动词过去式

句子时态为`过去时`时，动词要改成`过去时`，如`play`的过去式是`played`。

但是无论主语是什么，动词过去式的形式都相同。

> The kids <u>played</u> hide-and-seek yesterdy = 孩子们昨天玩捉迷藏

3. 过去分词

句子时态为`完成时`时，动词要改为`过去分词`，如played。

过去分词一般接在`助动词have/has`后面，而助动词得依主语不同选择使用have/has。第一人称用hava，第三人称用has。

> I `have played` tennis since I was eight = 我从八岁开始打网球
>
> Jason  `has played` baseball for eight years = 杰森已经打棒球八年了

4. 现在分词

句子时态为`进行时`时，动词要改为`现在分词`，一般在动词后面加`ing`。

现在分词一般接在`be动词am/is/are`后面，be动词需要依主语不同选择使用am/is/are。

第一人称I使用am，第二人称you或名词复数使用are，第三人称she/he/it使用is。

若是过去进行时 ，则be动词要改为过去式：`was或were`。

### 动词的变化规则

 英文中的动词变化，分为`规则变化`和`不规则变化`。

1. 规则变化

规则变化动词的过去时或过去分词，通常是在词尾加上`d`或`ed`。

2. 不规则变化

顾名思义，不规则变化动词的过去式及过去分词没有一定规则，但依然可以分为四种：

* A-B-C = 动词的三态皆不同
* A-B-B = 过去时和过去分词相同
* A-B-A = 现在时和过去分词相同
* A-A-A = 动词的三态皆相同

## 一般现在时-Present Simple Tense

一般现在时主要用来`表示人或事物的特点`，`经常或习惯性的动作`以及`现在的状况`；还可以`表示客观规律和永恒真理`等。句中常有often、always、from time to time等时间状语。

* 一般现在时的基本句型

> 主语 + be动词 + 宾语(主语补语) = 主系表
>
> 主语 + 实义动词 + 宾语 = 主谓宾

1. 使用be动词一般现在时句型时，be动词要视主语而定：

| 主语   | I    | you/we/they | he/she/it |
| ------ | ---- | ----------- | --------- |
| be动词 | am   | are         | is        |

2. 使用实义动词一般现在时的句型时，动词随主语变化：

| 主语                        | 动词             |
| --------------------------- | ---------------- |
| 第一人称单复数（I及we）     | 动词原型         |
| 第二人称单复数（you及you）  | 同上             |
| 第三人称复数（they）        | 同上             |
| 复数名词                    | 同上             |
| 第三人称单数（he、she、it） | 第三人称单数动词 |

* 单数动词变化规则

单数动词的变化规则，且发音也不同

1. 直接加`s`

| 词尾为清辅音，s发音为[s] | works,cooks,helps,asks   |
| ------------------------ | ------------------------ |
| 词尾为浊辅音，s发音为[z] | comes,reads,leaves,plays |

> She <u>likes</u> dogs = 她喜欢狗
>
> Mike <u>asks</u> a lot of questions = 迈克会问很多问题

2. 词尾为`o`，或词尾发音为[s,z]、[t,d]、[,]时，加`es`；若词尾已经有e，则直接加`s`即可。

|      | do->does，go->goes         |
| ---- | -------------------------- |
|      | misses,fixes,eases,teaches |

> John `goes` to school by buy = 约翰搭公交车上学
>
> Ms.Lin `teaches` History = 林老师教历史

3. 动词词尾为y时，去掉y加上`ies`。

| 词尾为y，去y加ies，发音为[z] | study-studies、carry-carries、fly-flies、cry-cries |
| ---------------------------- | -------------------------------------------------- |

> Peter studies hard = 皮特读书很用功
>
> The eagle flies high = 老鹰飞得很高

* 一般现在时形式

1. 肯定形式

主语 + 动词原形（如果主语是第三人称单数，动词要改为第三人称单数形式）

> I <u>wish</u> you every success = 祝你成功
>
> The picture <u>looks</u> very beautiful = 这副画看起来很美

2. 否定句式

* 谓语是系动词时则为`am/is/are + not`；
* 若谓语动词为行为动词，则在其加`don't`；
* 若主语为第三人称单数，则在动词原形前用`doesn't`。

> I am not student = 我不是一个学生
>
> Sometimes I do not know what to do = 有些时候我不知道该做什么才好
>
> He doesn't like playing computer games = 他不喜欢玩电脑游戏

3. 一般疑问句

* 谓语是系动词时则把系动词置于句首
* 若谓语动词为行为动词，用助动词do提问
* 若主语为第三人称单数，则用does提问，行为动词为原形

> Is she pretty? = 她漂亮吗？
>
> Do you like this movie? = 你喜欢这部电影吗？
>
> Does he speak Japanese? = 他说日语吗？

## 一般过去时-Past Simple Tense

表示在过去某个确定的时间发生的动作，存在的状态或习惯性动作，不强调对现在的影响，只说明过去。

常用的时间状语有：yesterday、last(week)、in(1945)、at that time、once、before、a few days ago、when等

> I sent a gift to my girlfriend last night = 我昨晚送了一份礼物给女友
>
> I arrived in Beijing yesterday

## 一般将来时-Future Simple Tense

表示在将来某个时间会发生的动作或情况。

常用的时间状语有：tomorrow、next day(week month year...)、soon、in a few minutes、by...、the day after tomorrow等

> He <u>will be</u> thirty next year = 他明年就30岁了
>
> I <u>will finish</u> my work soon = 我就要完成我的工作了
>
> I <u>shall go</u> to see my grandparents this evening = 我今晚要去看望我的祖父母

## 现在进行时-Present Continuous Tense

现在进行时表示当下的一段时间正在进行的某种活动或心里状态。

常用的时间状语有：now、at this time 等

> Don't disturb her. She is sleeping now. = 别打扰他，她正在睡觉。
>
> The girl is singing in her room at this moment. = 女孩这个时候正在她的房间唱歌

* 现在进行时的基本句型

> 主语 + be动词 + 现在分词(动词原形+ing) + ...
>
> It is raining outside = 外面正在下雨

* 否定句式

> 主语 + be动词 + not + 现在分词(动词原形+ing) + ...
>
> He's not fighting alone at this moment! = 此刻，他不是一个人在战斗！

* 一般疑问句形式

> 把系动词be置于句首，就会变成回答yes-no的疑问句。
>
> Are you surfing the Internet? = 你正在上网吗？

* 特殊疑问句

> 疑问词 + be动词 + 主语 + 现在分词（V-ing）？
>
> 疑问词 = (what/who/whom/where/how/when)

# 疑问句-Interrogative Sentences

疑问句主要用来提出问题。按其语法结构和交际功能可分为4种：`一般疑问句`、`特殊疑问句`、`选择疑问句`、`反意疑问句`。

> Are you a fireman? = 你是一名消防队员吗？： 一般疑问句
>
> What do you do at the weekend? = 你周末干什么？：特殊疑问句
>
> Would you like to have rice or noodles? = 你喜欢吃米饭还是苗条？：选择疑问句
>
> He is a Chinese teacher, isn't he? = 他是一名中文老师，不是吗？：反义疑问句

## 一般疑问句

一般疑问句常用来询问一件事情或一种情况是否属实，其答语一般是yes或no，或相当于yes或no的词语，因此这类问句又叫做“是非问句”。

* 一般疑问句的构成：句子谓语动词是`be动词`、`助动词`、`情态动词`时，把它们`放到主语前面`。

> Is there something wrong with this machine? = 这台机器有问题吗？
>
> Have you got today's milk = 你拿到了今天的牛奶吗？
>
> Shall we go on? = 我们继续向前吗？
>
> Can you speak French? = 你会说法语吗？
>
> Would you like me interpret for you? = 要不要我帮你翻译？

使用助动词时，原来的谓语动词要改为原形。

> Do you exercise? = 你做运动吗？
>
> Does Susan speak English? = 苏珊讲英语吗？
>
> Did your mon go shopping last weekend? = 你妈妈上个周末去买东西了吗？

* 一般疑问句的回答：

一般疑问句通常用简略形式回答。

> Have you got today's newspaper? = 你有今天的报纸吗？
>
> Yes, I have. = 是的，我有
>
> No, I haven't = 不，我没有

* 一般疑问句的否定结构

表示惊奇、责怪、建议、看法等，只要将not置于主语之后或将not放到主语之前与be，have等助动词或情态动词结合在有一起就行。

> Will he not agree with you? = 他不同意你的观点吗？
>
> Isn't you father a Party member? = 你父亲不是党员吗？
>
> Haven't you any sister? = 你没有姐妹吗？
>
> Don't you like this movie? = 你难道不喜欢这部电影吗？

这种否定结构的疑问句的回答与汉语的习惯不同。如果`回答是肯定的`，就用`yes+肯定结构`；如果`回答是否定的`，就用`no+否定结构`。(情况与反意疑问句类似)

> Can't he afford a new hourse? = 他买不起新房吗？
>
> Yes, he can = 不，他能。
>
> No, he can't = 是的，他不能。

## 特殊疑问句

特殊疑问句是对句子的某一特殊部分提出疑问，通常以who、whoes、what、which、where、when、why、how等疑问词开头。因此又叫做`wh-问句`。

* 常用的疑问代词和疑问副词

疑问代词：who、whom、whose、which、what

疑问副词：when、where、why、how

how构成的短语：how many 多少个（独立用；或可数名词复数）、how much 多少（独立用；或跟不可数名词）、how old 多大年纪、how far 多远、how often 多久等。

* what问的是“事物”

`what`指`什么`，可以用来问“姓名”、“职业”、“动作”、“事件”、“物品”。想问的具体事项可以从表语（补语）或动词得知。

> What is you <u>name</u>? = 你的名字叫什么？
>
> What are <u>you</u> doing? = 你正在做什么？
>
> What does you father do? = 你的父亲是做什么的？
>
> What do you want? = 你想要什么？

what后面也可加上名词，说明想问的具体事项。

> What + 名词 + be动词/助动词 + 主语 + 补语/动词？

> What fruits are good for eyes? = 什么水果对眼睛有益？
>
> What colors do you like? = 你喜欢什么颜色？

问星期和日期的方法

> What day is it? = 今天是星期几？ it's Monday = 今天是星期一
>
> What date is it? = 今天是几月几日？it's March 27 = 今天是三月二十七日

* where问的是“地点”

`where`指`何处`，可以用来问“地点”、“方向”、“位置”等等。

> Where is your mom? = 你的妈妈在那里？
>
> Where did you put it? = 你把它放在哪儿了？

* when问的是“时间”

`when`指`何时`，可以用来问“时间点”。 小至几点几分，大至月份、季节、年份等，都是可以询问的时间范围。

> When will you be there? = 你何时会到那里？
>
> When did the meeting begin? = 会议是何时开始的？

when所询问的时间范围很广，因此答案也因人而异。

> When did you see him? = 你何时看到他的
>
> I saw him this morning = 我今天早上看到他的
>
> I saw him two hours ago = 我两个小时前看到他的

如果想问出`具体的时间`，可以用`what或which+时间名词`来将问题具体化：

> `What time` did you go to bed? = 你几点上床睡觉
>
> `Which/What year` were you born in? = 你是哪一年出生的？
>
> `What date` is your birthday? = 你的生日是几月几日？

* who问的是“身份”

`who`指`谁`，可以用来问“身份”、“对象”、“关系”，who可以用来作主语，也可以用来做宾语。

> Who is that? = 那个人是谁？
>
> Who can drive? = 谁会开车
>
> Who took my umbrella? = 谁拿走了我的雨伞？

* why问的是“原因”

`why`指`为什么`，通常用来问“原因”。

> Why are you late? = 你为什么迟到？
>
> Why didn't he come to the party? = 他为何没来参加派对

* how问的是“方式”

`how`指`如何`，可以用来问“方法”，也可以用来问候对方“近况”，或事情进行的“状况”。

> How do you go to work? = 你怎么去上班的？
>
> How are you? = 你好吗？
>
> How are you doing? = 你过得好吗？
>
> How have you been? = 你过得好吗？
>
> How's it going? = 一切都好吗？

how后面常接不容的形容词，用来询问不同的事物：

> how old 多老、how many 多少、how much 多少钱、how long 多久等等

* which问的是“哪一个”

`which`指`哪一个`，通常用来做“有选择性的”提问。which后面常接名词，让问句更加明确。

> Which is you car? = 你的车市哪一辆？
>
> Which color do you like beter? = 你比较喜欢哪一个颜色？

# 连词-Conjunctions

连词是一种虚词，在句子中起连接作用，可以用来连接单词、短语、从句，使句子能够传达更完整的意思。

连词按其功能及用法分为`对等连词（并列连词）`、`相关连词（关联连词）`、`从属连词`。

## 并列连词

并列连词用来连接彼此是并列关系的单词、短语、分局或从句，被连接的两个部分彼此是独立的，在句法上是平等的。

常用的并列连词有and、or、but、for、not only... but(also)...、both...and...等。

* and（和；又；而且）：连接同性质或语义的单词、短语、或从句

> Peter is handsome and rich = 彼得英俊又有钱
>
> Jennifer and Sarah are twins = 珍妮弗和萨拉是双胞胎

* but（但是；却）：连接语义相反或矛盾的单词、短语或从句。

> The queen is beautiful but evil = 皇后美丽却恶毒
>
> This is a very creative but not so practical idea = 这是个非常有创意却不太实际的点子

* yet（而；然而）：与but用法一样，都是连接语义相反或矛盾的单词、短语或从句

> The journey was tiring yet wonderful = 旅程累人却美好
>
> The math question was difficult, yet he solved it = 这数学题很难，但他解出来了

* so（因此；所以）：连接表示“结果”的从句，用来表示因果关系

> He was sick yesterday, so he didn't go to school = 他昨天生病了，所以没有去上学
>
> There is a typhoon coming, so we cancel the picnic = 台风要来了，所以我们取消了野餐

* for（因为；为了）：连接表示“原因的从句”，用来提供“理由”

> She didn't go to the party, for she wasn't invited = 她没有去派对，因为她没有收到邀请

* or（或者；还是）：连接两个词性或性质相同的单词、短语或句子，提供两种不同的选择

> 连接两个名词 - Do you want to go by car or subway = 你想开车还是搭地铁去
>
> 连接两个性质相同的句子 - We can visit a near country, or we can just have a short trip to the countryside = 我们可以去一个邻国玩，也可以到乡下短途旅行

or也可以表示“否则”，用来连接“带有警告意味”的句子

> Don't stay up too late, or you'll oversleep yourself tomorrow = 别熬夜到太晚，否则你明天会睡过头

* either... or...

or常与either连用，表示“两者其中之一”或“不是...就是...”

either... or... 可连接相同词性的单词或短语。当连接两个主语是，动词随or后面的主语做变化。

> We can either go to the movies or go shopping = 我们可以去看电影，或是去逛街
>
> Either he or I <u>am</u> going with her = 不是他就是我会跟着她一起去
>
> ​						be动词随or后面主语I变化

* nor（也不）：用在有否定意味的句子中，配合前面有否定意味的单词、短语或句子（含no，not，never等单词），连接具有相同词性的单词或短语，以表示“既不...也不”，“既没...也没”。

> My new neighbor is not polite nor friendly = 我的新邻居既没礼貌，也不友善
>
> The man has no family nor any friends = 那男子既没家人也没有朋友

* neither.. nor...

nor常与neither连用，表示“两者皆非”或“既不...也不”。

neither.. nor...可连接相同词性的单词或短语。连接两个主语时，动词随nor后面的主语做变化。

> She will marry neither Andy nor John = 她即不会嫁给安迪，也不会嫁给约翰
>
> Neither Mom nor Dad was at home at that time = 妈妈跟爸爸当时都不在家

## 从属连词

从属连词用来连接两个或两个以上的分句，形成复杂句中的从属分句，即引导名词性从句和状语从句。

常用的从属连词有that、whether、when、although、because等。

当一个独立的句子所表达的意思不够完整，需要更多说明时，就可以用连词引导一个从句来修饰主句，为主句补充说明“时间”、“原因”或“条件”等。

* 从属连词的基本句型

> 主句 + 从属连词 + 从句 = 从属连词 + 从句，主句

从属连词的位置，可以放在句中，可无缝连接前后两个句子；也可以放在句首，引导出从句后，以“逗号”与主句隔开

> I was terrified when the earthquake occurred = When the earthquake occurred, I was terrified
>
> 地震发生时我吓坏了

* 什么时候用从属连词

1. 必须提供一个跟“时间”有关的概念是

> We cleaned up the house <u>before</u> Mon came back = 我们在妈妈回来之前要把屋子打扫干净

2. 需要说明“原因”时

> She didn't go to school <u>because</u> she was ill  = 她没去上学是因为她病了

3. 用来表示“目的”时

> They practiced hard <u>in order to</u> win the game = 他们练习得很努力是为了赢得比赛

4. 说明事件发生的“条件“时

> I will not go to the zoo <u>if</u> it rains = 如果下雨，我就不会去动物园

5. 表示”让步“时

> <u>Although</u> we are neighbors, we never talk to each other = 虽然我们是邻居，却从来没跟彼此说过话

## 使用的误区



# 介词-Prepositions

介词是一种虚词，又称`前置词`，通常置于`名词、代词或相当于名词的结构`之前，在句子中不能单独作句子成分，只表示其后的名词或相当于名词的词语或句中其他成分之间的关系。

# 数词-Numerals

表示`数目多少或顺序先后`的词叫做数词。数词与不定代词很相似，其用法相当于名词与形容词的用法。

数词大体分为两种，一种是`基数词`，即表示数目多少的数词，如one、two、three、thirty等。另一种是`序数词`，如first、second、tenth等。

# 量词-Quantifiers

用来表示`人、事物或动作的数量单位`的词，叫做量词。英语中有的量词有很多种意思，它们没有像汉语中那样的量词分类。量词的翻译需要大量积累，以及对中西文化的理解。

# 感叹词-Interjections

感叹词用来`表示说话人的各种感情`，如欢喜、痛苦、惊恐、赞叹等。一般置于句首，感情强烈时，句尾用”!“感叹号来表示。

感叹词不构成后面所接句子的语法成分，却在意义上与它关联，后面的句子一般说明这种情绪的性质、原因。

感叹词是英语口语中最富有表现力的词语之一，感叹词一般用于口语之中，是人们日常交际中常用的句子。

# 被动语态-Passive Voice

主动语态的主语是句子中动作的”执行者“，而被动语态的主语，则是句子中动作的”承受者“。

被动语态是由`助动词be + 及物动词的过去分词`构成。助动词be随主语的人称、数、时态和语气的不同而变化。

如果要强调动作的执行者，则用介词by引出的短语来表示。

> 主动：Peter cleaned the bedroom = 彼得清理了房间
>
> 被动：The bedroom was cleaned (by Peter) = 房间被彼得清理了

# 语气类别-Moods Classification

语气是用来表示说话人对某一行为或事情的看法和态度。

* 陈诉语气

陈述语气是`表示说话人认为他所说的话是一个实事`，谓语动词可以用各种时态。一般用于陈诉句、疑问句和部分感叹句。

> Luxun is famous for his articles = 鲁迅以他的文章而闻名于世

* 祈使语气

祈使语气是`表示说话人向对方提出劝告、命令、希望或禁止等`，一般用动词原形，主语常常被省略。

1. 祈使语气表示祈愿

> Have a good day, sir! = 祝你愉快，先生！
>
> May you have a happy new year = 祝你新年快乐

2. 祈使语气表示恳求、建议或邀请

> Speak more slowly, please = 请讲慢些
>
> Make yourself at home = 请随便，不要客气

3. 祈使句以do开头，则表示强调

> Do give me a call = 一定要给我打电话
>
> Do be quiet a moment = 一定要安静一会儿

4. 否定祈使句的构成通常为don't + 动词原形，但有时也用never + 动词原形构成

> Don't be so nervous! = 别那么紧张！
>
> Never do that again! = 别在那么做了！

5. 肯定祈使句的被动语态：Let + 宾语 + be + 过去分词

> 主动语态：Close the door = 关上门
>
> 被动语态：Let the door be closed = 把门关上

6. Let + 宾语 + 动词原形 + 其他部分，构成祈使语气，通常用于第一和第三人称的祈使句中

> Let me look at map = 让我看看地图
>
> Let us have something iced to drink = 让我们喝点冷饮把

* 虚拟语气

虚拟语气是一种特殊的动词形式，用来`表示说话人的主观愿望、假想、怀疑、猜测、建议等`含义，所表示的不一定是事实，可能与事实相反。

> 表示假设：If I were not so busy,I would go with you  = 假如我不太忙，我会和你一起去
>
> 表示愿望：I wish my father were here. He would know how to fix this car = 我希望我爸爸在这儿就好了，他应该知道怎样把这车修好
>
> 表示建议：He suggests that we should all go home = 他建议我们都回家

# 句子成分-Sentence Constituents

`组成句子的各个部分`叫做句子成分。句子成分包括：`主语、谓语、宾语（直接宾语和间接宾语）、宾语补足语、表语、定语和状语`。

* 主语

主语是`句子所要说明的人或事物`，是句子的`主体`。一般由`名词、代词、不定式或相当于名词的词或短语`来充当，位于句首。

> 名词作主语：<u>The desk</u> is old = 这张课桌旧了
>
> 代词作主语：<u>He</u> is a doctor = 他是一名医生
>
> 不定式作主语：To see is to believe = 眼见为实
>
> 名词短语作主语：<u>The secret of success</u> is to start from scratch and keep on scratching = 成功的秘诀在于从磨炼开始，并坚持不断地磨炼

* 谓语

谓语是用来说明主语`干什么`或`怎么样`的。谓语或谓语部分里主要词必须用`动词`。在运用中，谓语和主语在人称和数的方面应该保持一致，且谓语应在主语后面。

> 实义动词作谓语：I <u>play</u> basketball in the morning = 我早上踢球
>
> 情态动词和实义动词作谓语：We `can speak` French = 我们会讲法语

* 宾语

表示`及物动词的动作对象或内容`，通常位于动词之后或用于介词后构成介词短语。一般由名词、代词、不定式或相当于名词的词或短语来充当。

> 代词和名词作间接宾语和直接宾语：Pass <u>me</u> the <u>book</u>, please = 请把这本书给我
>
> 不定式作宾语：I want to eat <u>two hamburgers</u> now = 不定式作宾语
>
> 动名词作宾语：She likes <u>skipping</u> = 她喜欢跳绳

* 表语

表语是用来`说明主语的身份、特征、性质或状态`的词。常位于`系动词之后`，由名词、形容词、副词、介词短语或不定式来充当。

> 名词作表语：It is a <u>flower</u> = 这是一朵花
>
> 介词短语作表语：We are <u>at work</u> = 我们在上班
>
> 形容词作表语：She is <u>nice</u> = 形容词作表语
>
> 不定式作表语：Her job is <u>to look after</u> her baby = 她的工作就是照顾她的孩子

* 定语

定语是用来`修饰名词或代词`的。定语的位置很灵活，凡是有名词、代词的地方都可以有定语。通常名词、形容词、数词、分词、不定式、介词短语或定语从句都可以用作定语。

> 名次作定语：The boy pupil is in Class Three = 这个男生在3班
>
> 介词短语作定语：The girl <u>in red</u> is my sister = 穿红衣服的女孩是我的妹妹

* 状语

状语是用来`修饰动词、形容词和副词`的句子成分。

状语表示`动作、行为发生时的时间、原因、地点、方式、目的或程度`等。

一般由副词、不定式、介词短语或相当于副词的词或短语来充当。状语通常位于句末，但有时也可以放在句首或句中。

> 副词作状语：I can jump <u>high</u> = 我跳得高
>
> 介词短语作状语：<u>Without the idea</u>, I can't finish the work = 没有这个创意，我不可能完成这份工作

* 补语

补语是用来对句子中的某些成分（`主语、表语、宾语等`）进行`补充说明`。补语的形式有动词的类别来决定。

> 主语补语：<u>Tom</u> was made <u>monitor</u> = 汤姆被选为班长
>
> 宾语补语：I made <u>Tom monitor</u> = 我选汤姆当班长
>
> 表语补语：I am <u>sure</u> <u>to succeed</u> = 我肯定能成功

通常情况下，主语和宾语前的成分是定语，谓语前的成分是状语，时间词作状语放在句子后面。

> （定语）	主语（状语）谓语	（定语）	宾语	（状语）
>
> （The tall）boy（often）goes（to the big）zoo（on Sunday）
